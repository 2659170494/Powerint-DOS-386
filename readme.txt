这是我和@min0911_花了2个月写出来的操作系统。
目前功能还不太完善，希望见谅。
系统基本信息：
基于Haribote的工具编译而成，可以使用GCC编译，但仅仅支持HRB和COFF，不支持ELF。
保护模式，段试内存管理，单任务系统，最低内存4MB。
和DOS系统很接近，基本命令和DOS系统无太大区别。
实现VGA显示模式切换，ACPI电源管理，IDE硬盘读写，ASM编译器，FAT12文件系统管理，程序应用运行，
BATCH批处理文件运行，BMP位图浏览，中文文档浏览，支持PS/2鼠标，计时器，类GRUB引导多系统，等功能。
但因为没实现多任务系统保护，所以系统容易受到破坏。
内存分页也是个要填的坑。
话说一个DOS系统没必要进长模式吧（笑）。
